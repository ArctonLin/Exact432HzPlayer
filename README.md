# Exact 432Hz Player (Android APP)

### Google Play
[<img src="assets/google-play-badge.png" height="80">](https://play.google.com/store/apps/details?id=com.exact432hz.exact432hzplayer)

### Introduction
Exact 432Hz Player analyzes the instrument frequencies from the audio file (PCM raw data) and automatically adjusts the playback pitch to convert all music to a 432Hz tone.

This APP is based on Retro Music Player with GPL-3.0 license. Thanks for the developers/contributors of RetroMusicPlayer project.
https://github.com/RetroMusicPlayer/RetroMusicPlayer
